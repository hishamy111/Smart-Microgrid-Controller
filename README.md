## Overview

The Smart Microgrid Controller is an ESP32-based embedded system designed to monitor and automatically manage renewable energy sources.

The controller continuously monitors solar, wind, and battery power, automatically selects the most suitable energy source, controls battery charging, and provides real-time monitoring through a built-in web dashboard.

---

## Features

- Solar Monitoring
- Wind Monitoring
- Battery Monitoring
- Automatic Source Switching
- Battery Charging Control
- Relay Control
- Real-Time Web Dashboard
